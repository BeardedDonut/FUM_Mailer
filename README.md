# FUM_Mailer
add a config.json file in Server folder and it should contain json data below:
-------------config.json Template
{
  "Database" :
  {
        "user" : "<database_user>" ,
        "password" : "<password>" ,
        "host" : "<database_host_address>" ,
        "database" : "<database_name>"
  },

  "Email" :
  {
        "email" : "<mail_account>" ,
        "password" : "<mail_password>"
  }

}
------------
